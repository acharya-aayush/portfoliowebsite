# Blog Images

Place your blog post images here.

## Naming Convention

Use descriptive names:
- ✅ `chitragupta-dashboard.png`
- ✅ `ocr-preprocessing-pipeline.jpg`
- ✅ `api-architecture-diagram.png`
- ❌ `img1.png`
- ❌ `screenshot.jpg`

## Image Specs

- **Max width:** 1200px
- **Formats:** PNG (screenshots), JPG (photos), WebP (best compression)
- **Compress:** Use TinyPNG or similar before uploading
- **Alt text:** Always describe the image in markdown

## Usage in Blog Posts

```markdown
![Alt text describing the image](/blog/images/your-image.png "Optional title on hover")
```

## Example

In your blog post:
```markdown
## The Dashboard

Here's what the interface looks like:

![Chitragupta AI Dashboard showing extracted form fields](/blog/images/dashboard-screenshot.png "Main dashboard with OCR results highlighted")
```

This will render the image from `public/blog/images/dashboard-screenshot.png`
